# FoxTail
Rails View Components built with Flowbite and Tailwind CSS

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "fox_tail"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install fox_tail
```
